# qt-photoalbum

qt photoalbum — приложение на Qt для просмотра изображений из папки с возможностью переключения между файлами.

## О проекте

Приложение позволяет загружать изображения из выбранной директории и просматривать их по одному, переключаясь между файлами с помощью кнопок навигации.
Изображения могут загружаться как из файловой системы, так и из ресурсов Qt (`.qrc`).

## Возможности

- Просмотр изображений из папки
- Переключение между изображениями (вперёд / назад)
- Автоматическое обновление текущего изображения
- Блокировка кнопок на первом и последнем изображении
- Работа с ресурсами Qt (`.qrc`)
- Простой и понятный графический интерфейс

## Используемые технологии

- C++
- Qt (Qt Widgets)
- Qt Designer
- `QLabel` для отображения изображенийй
- `QDir` для работы с файловой системой
- Система ресурсов Qt (`.qrc`)

Основная логика:
- хранение текущей папки
- хранение текущего индекса изображения
- обновление изображения при переключении
- управление доступностью кнопок

## Как работает

1. Приложение загружает изображения из папки
2. Отображает первое изображение
3. Пользователь переключает изображения кнопками:
   - влево — предыдущее
   - вправо — следующее
4. Кнопки автоматически блокируются на границах списка
